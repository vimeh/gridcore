Gridcore Error Handling Test

Open the gridcore app and test the following formulas:

Test Cases:

  1. Invalid Reference (#REF!):
    • Enter: =XYZ999
    • Expected: Should display #REF! error
  2. Unknown Function (#NAME?):
    • Enter: =UNKNOWNFUNC(A1)
    • Expected: Should display #NAME? error
  3. Type Mismatch (#VALUE!):
    • Enter: ="text" + 5
    • Expected: Should display #VALUE! error
  4. Circular Reference (#CIRC!):
    • In A1, enter: =B1
    • In B1, enter: =A1
    • Expected: Both should display #CIRC! error
  5. Division by Zero (#DIV/0!):
    • Enter: =5/0
    • Expected: Should display #DIV/0! error
