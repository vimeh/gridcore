Manual Error Testing for Gridcore

Test 1: Invalid Reference

Formula: =XYZ999

✅ Expected: Error message containing "Invalid"

The cell should display "Invalid cell reference" and the error should appear in the error display

Test 2: Unknown Function

Formula: =UNKNOWNFUNC(A1)

✅ Expected: Error message containing "Unknown function"

The cell should display "Unknown function: UNKNOWNFUNC"

Test 3: Type Mismatch

Formula: ="text" + 5

✅ Expected: Any error display

The concatenation operator should work, but if not, an error should be shown

Test 4: Circular Reference

In E1: =F1+1

In F1: =E1+1

✅ Expected: Error message containing "Circular"

Both cells should display "Circular reference detected"

Test 5: Division by Zero

Formula: =1/0

✅ Expected: Error message containing "DIV" or "Division"

The cell should display "Division by zero"

Test Steps:

  1. Open Gridcore App
  2. Enter each formula above in different cells
  3. Verify that errors are displayed in the error container (top-right)
  4. Verify that cells show error values
  5. Verify that the formula bar still shows the original formula
  6. Verify that you can re-edit cells with errors
